BROWSER

A browser is an application, software program, that is needed to access websites on the internet. The browser will take the information sent to it from a web server and present the information to the user. This is most often done with the help of HTTP, Hypertext Transfer Protocol. The protocol can be described as a language that both the browser and web server understands. By clicking on a link or writing an address to a website the users browser will send a request to the connected web server and the web server will respond to the request.

A few examples of browsers are: Google chrome, Safari, Mozilla firefox and Internet explorer.

There are several types of extensions that can be added to a browser making it more customised for the user, anything from ad blockers to extensions that search for discount codes when shopping online. One could say that they are both the interpreters and butlers of the world wide web.
